On this page you’ll find a quick reference to common HTML terms and a table of frequently used named character entities.

HTML Terms

Attribute
An HTML element property that provides additional information or functionality for an element.
Block-Level Element
HTML tags that create blocks of content such as headings and paragraphs.
Child Element
Any HTML element contained inside another HTML element.
Closing Tag
Appears after an element’s content and indicates the end of an element.
Comment
Reference information added to the code for explanation or notes.
CMS
Content Management System used to create and manage web content.
CSS
Cascading Style Sheets used to control presentation and layout of web pages.
Doctype
Document Type Declaration that identifies the version of HTML used.
DOM
Document Object Model representing the structure of HTML documents.
Elements
HTML elements consist of tags and the content they contain.
FTP
File Transfer Protocol used for transferring files between servers.
HTTP
Hypertext Transfer Protocol for transmitting HTML documents.
Inline-Level Elements
Elements that appear within normal text flow.
JavaScript
Scripting language used to control functionality and behavior on web pages.
Metadata
Additional information about a document, typically stored in meta tags.
MIME Type
Multipurpose Internet Mail Extension used to define file types.
Parent Element
An HTML element that contains another element.
Replaced Element
An element where content is replaced by external content such as images or video.
Tag
Markup characters that indicate what type of content appears on the page.
URL
Uniform Resource Locator, the address of a resource on the web.
W3C
World Wide Web Consortium responsible for web standards.
